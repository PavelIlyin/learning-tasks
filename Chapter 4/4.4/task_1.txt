Каким будет результат при обращении к свойству объекта ref? Почему?

function makeUser() {
  return {
    name: "John",
    ref: this
  };
}

let user = makeUser();

alert( user.ref.name ); // Каким будет результат? 

Ответ: 
TypeError, свойства не удаcтся прочитать (undefined).
В момент вызова значение this не смотрят на объявление объекта.
Значение this одно для всей функции.
